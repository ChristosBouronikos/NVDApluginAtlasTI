## What is the NVDA key?

The **NVDA key** is a modifier key used for NVDA commands.
Default: **Insert** or **Numpad Insert** (Caps Lock can be enabled in settings)

Example: `NVDA+Alt+D` means: Hold Insert + Alt, press D

---

## Features

- Quick panel navigation
- Reading helpers for codes, quotations, documents
- Improved control labels
- Greek/English support

## Shortcuts

### Panel Navigation
| Shortcut | Action |
|----------|--------|
| `NVDA+Alt+D` | Documents |
| `NVDA+Alt+C` | Codes |
| `NVDA+Alt+Q` | Quotations |
| `NVDA+Alt+M` | Memos |
| `NVDA+Alt+P` | Navigator |

### Reading
| Shortcut | Action |
|----------|--------|
| `NVDA+Shift+C` | Code details |
| `NVDA+Shift+Q` | Quotation text |
| `NVDA+Shift+D` | Document info |
| `NVDA+Shift+P` | Current panel |
| `NVDA+Shift+H` | Help |

## Installation

1. Download `.nvda-addon`
2. Double-click to install
3. Restart NVDA

## Troubleshooting

- **Not loading:** Check NVDA log (`NVDA+N` → Tools → View Log)
- **Shortcuts not working:** Ensure Atlas.ti has focus
