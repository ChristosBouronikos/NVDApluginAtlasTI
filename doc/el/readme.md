## Τι είναι το πλήκτρο NVDA;

Το **πλήκτρο NVDA** είναι ένα πλήκτρο τροποποίησης για τις εντολές του NVDA.
Προεπιλογή: **Insert** ή **Numpad Insert** (το Caps Lock μπορεί να ενεργοποιηθεί)

Παράδειγμα: `NVDA+Alt+D` σημαίνει: Κρατήστε Insert + Alt, πατήστε D

---

## Χαρακτηριστικά

- Γρήγορη πλοήγηση πινάκων
- Βοηθοί ανάγνωσης για κωδικούς, αποσπάσματα, έγγραφα
- Βελτιωμένες ετικέτες ελέγχων
- Υποστήριξη Ελληνικών/Αγγλικών

## Συντομεύσεις

### Πλοήγηση Πινάκων
| Συντόμευση | Ενέργεια |
|------------|----------|
| `NVDA+Alt+D` | Έγγραφα |
| `NVDA+Alt+C` | Κωδικοί |
| `NVDA+Alt+Q` | Αποσπάσματα |
| `NVDA+Alt+M` | Σημειώσεις |
| `NVDA+Alt+P` | Περιήγηση |

### Ανάγνωση
| Συντόμευση | Ενέργεια |
|------------|----------|
| `NVDA+Shift+C` | Κωδικός |
| `NVDA+Shift+Q` | Απόσπασμα |
| `NVDA+Shift+D` | Έγγραφο |
| `NVDA+Shift+P` | Πίνακας |
| `NVDA+Shift+H` | Βοήθεια |

## Εγκατάσταση

1. Κατεβάστε το `.nvda-addon`
2. Διπλό κλικ για εγκατάσταση
3. Επανεκκίνηση NVDA

## Αντιμετώπιση Προβλημάτων

- **Δεν φορτώνει:** Δείτε το αρχείο καταγραφής (`NVDA+N` → Εργαλεία → Αρχείο Καταγραφής)
- **Συντομεύσεις:** Βεβαιωθείτε ότι το Atlas.ti έχει την εστίαση
