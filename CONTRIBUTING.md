## How to Contribute / Πώς να Συνεισφέρετε

### Reporting Issues / Αναφορά Προβλημάτων

1. Check if the issue already exists / Ελέγξτε αν το πρόβλημα υπάρχει ήδη
2. Create a new issue with:
   - NVDA version
   - Atlas.ti version
   - Windows version
   - Steps to reproduce
   - Expected vs actual behavior

### Suggesting Features / Πρόταση Χαρακτηριστικών

Open an issue describing:
- What accessibility problem you're facing
- How the feature would help
- Any ideas for implementation

### Code Contributions / Συνεισφορά Κώδικα

1. Fork the repository
2. Create a feature branch
3. Make your changes
4. Test with NVDA and Atlas.ti
5. Submit a pull request

### Guidelines / Οδηγίες

- Follow existing code style
- Add comments for complex logic
- Update documentation if needed
- Test on Windows 10/11 with NVDA

### Testing / Δοκιμές

Before submitting:
1. Install the addon in NVDA
2. Test all shortcuts work
3. Test with Atlas.ti open
4. Check NVDA log for errors
